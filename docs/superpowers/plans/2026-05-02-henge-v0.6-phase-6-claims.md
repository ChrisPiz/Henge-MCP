# Henge v0.6 — Phase 6: Claim verification

> **For agentic workers:** REQUIRED SUB-SKILL: superpowers:subagent-driven-development.

**Goal:** Después de Haiku sintetizar el consensus, Sonnet 4.6 extrae N claims falsables del texto. gpt-5 cross-lab verifica cada claim contra los 9 frame outputs: qué frames lo sostienen, qué frames lo contradicen, qué fuerza tiene el soporte. Cada claim del consensus se anota inline en el reporte: `[supported by N/9]` / `[contested: frame_X]` / `[unsupported]`.

**Architecture:**
- Nuevo módulo `henge/claims.py` con dos funciones públicas (`extract_claims`, `verify_claims`) + dataclasses.
- Server.py hook después de `synthesize_consensus`: extract → verify → surface en response.
- HTML viz: nuevo bloque al lado del consensus con la lista de claims y su soporte. (No anotación inline — más simple, más legible.)
- Feature flag `HENGE_ENABLE_CLAIM_VERIFICATION` (default `true`).
- Cross-lab: extracción Sonnet (Anthropic), verificación gpt-5 (OpenAI). Si Haiku alucinó un claim que ningún frame dijo, gpt-5 lo detecta porque no comparte afinidad de output con Haiku.

**Tech Stack:** providers registry, dataclasses, asyncio, JSON parsing, viz extensions.

### Task 6.2: `extract_claims(consensus_text)` (Sonnet)

System prompt: "Extract N falsifiable claims from the consensus. Each must be a single proposition. Tag each as factual / prescriptive / causal."

Output JSON: `[{text: str, type: str}, ...]`

Use `complete("anthropic/sonnet-4-6", req)`. `max_tokens=2000`. JSON tolerant parsing.

### Task 6.3: `verify_claims(claims, nine_outputs)` (gpt-5)

System prompt: "For each claim, identify which frames (by name) support it, which contest it. Rate support_strength: strong (≥6/9 explicitly), moderate (3-5/9), weak (1-2/9), unsupported (0/9 = consensus hallucinated)."

Output JSON: `[{claim_text, claim_type, supporting_frames, contesting_frames, support_strength}, ...]`

Use `complete("openai/gpt-5", req)` with `reasoning_effort="low"`, `max_tokens=4000`. JSON tolerant parsing.

### Task 6.4: Wire into server.decide()

After `synthesize_consensus`:
```python
claims = []
verifications = []
extract_usage = None
verify_usage = None
if ENABLE_CLAIM_VERIFICATION and consensus_text:
    claims, extract_usage = await extract_claims(consensus_text)
    if claims:
        verifications, verify_usage = await verify_claims(claims, successful_frames)
```

Surface in BOTH success response paths (line ~513 full response and line ~647 compact response):
```python
"consensus_claims": [
    {
        "claim_text": v.claim_text,
        "claim_type": v.claim_type,
        "supporting_frames": v.supporting_frames,
        "contesting_frames": v.contesting_frames,
        "support_strength": v.support_strength,
    }
    for v in verifications
],
```
